Nous divisons le projet en 2 grandes parties: (1) Algorithmes de coloriage d'un graphe statique (2) Applications permettant de colorier un graphe dynamique et de résoudre une grille du Sudoku

1. Algorithmes de coloriage d'un graphe statique:

Dans cette partie, il y a 2 sections plus petites, y compris les algorithmes de coloriage des graphes statiques et les algorithmes de coloriage des graphes dynamiques.

Pour voir le code détaillé et le résultat de chaque algorithme, vous ouvrez le fichier graph_coloring_algo_code.ipynb à l'aide du Visual Studio Code, du Pyzo ou du Jupyter Notebook et exécutez chaque algorithme à son tour. Une autre façon de voir le code détaillé et le résultat de chaque algorithme est d'ouvrir le fichier PDF graph_coloring_algo_code.

Pour voir le pseudo-code de chaque algorithme, vous ouvrez le fichier PDF final_report et recherchez chaque algorithme.

2. Applications permettant de colorier un graphe dynamique et de résoudre une grille du Sudoku

Dans cette partie, il existe 2 applications distinctes, y compris le dossier graph_coloring_app et le dossier suduku_app.

Pour voir le résultat d'une application permettant de colorier un graphe dynamique, vous ouvrez le dossier graph_coloring_app et exécutez le fichier main.py à l'aide du Visual Studio Code ou du Pyzo. Pour colorier un graphique, vous cliquez sur Aidez-Moi pour voir les consignes.

Pour voir le résultat d'une application permettant de résoudre une grille du Sudoku à l'aide d'un algorithme de coloration de graphe, vous ouvrez le dossier suduku_app et exécutez le fichier main.py. Pour résoudre un Sudoku, vous cliquez sur Résoudre. Pour réinitialiser le jeu ou pour recommancer, vous cliquez sur le Recommancer.

We divide the project into 2 big parts: (1) graph coloring algorithms (2) graph coloring and sudoku applications

1. For graph coloring algorithms:

In this part, there 2 smaller sections including Graphes Statiques and Graphes Dynamiques.

To see the detailed code and result of each algorithm, open the graph_coloring_algo_code.ipynb file using Visual Studio Code, Pyzo or Jupyter Notebook and execute each algorithm in turn. Another way to see the code and the result is to open the graph_coloring_algo_code pdf file.

To see the pseudo-code of each algorithm, open the final_report pdf file and search for each algorithm. 

2. For graph coloring and sudoku applications

In this part, there are 2 smallers applications including graph_coloring app folder and suduku_app folder.

To see the result of a graph coloring application, open the graph_coloring app folder and run the main.py file using Visual Studio Code or Pyzo. To color a graph, click on Aidez-moi to see the instrutions.

To see the result of a sodoku application using a graph coloring algorithm, open the suduku_app folder and run the main.py file. To resolve a sudoku, click on Résoudre. To reset the game, click on Recommancer.
